AVLTree

Implementation of an AVL Tree. This is a self-balancing tree with fast search, insert and delete complexities of O(log(n)). In our implementation we don't save the 'parent' of a node in the node but retrieve the parent by traversing the tree. This saves us the O(n) space-complexity in favour of the O(log n) time-complexity.
